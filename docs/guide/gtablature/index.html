Guido extension for guitar tablatures.

Since version 1.7.7, the Guido engine supports guitar tablatures.

Designing a tablature for guitar

Two extensions are provided to support a special staff for tablature : - the \clef tag supports a new 'TAB' type - the \staffFormat tag supports a 'TAB' type.

Tablatures are using special notes names that behave like regular notes.

ComplexDiagram( Sequence ('s', ':', NonTerminal('string'), ':', NonTerminal('fret')) ).addTo()

Example

d1*3/4  c#0/8 h&/4  _/4.  c&&/2. cis/4.. fa##1/4 sol&0
